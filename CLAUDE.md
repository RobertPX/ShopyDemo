# Nexvo Showcase — Instrucciones de Proyecto

Este proyecto sigue el documento de arquitectura completo: @ARCHITECTURE.md

Reglas fijas para todas las sesiones:
- No implementes pagos reales ni conexión a backend a menos que se indique explícitamente.
- No avances a una fase del roadmap (sección 12 de ARCHITECTURE.md) sin que el usuario lo pida.
- Sigue la convención de nombres de archivos y carpetas exacta de la sección 11.
- Todo color en componentes debe usar las variables CSS de la sección 5, nunca un hex directo.
- Antes de instalar una librería que no esté en la sección 3, pregunta primero.
